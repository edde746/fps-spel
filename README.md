# fps-spel
Första Unity 3D projektet, fps spel, skjut alla måltavlorna innan tiden tar slut

## Features
* Movement
* Enkelt vapen med "begränsad" ammuntion
* Tidsbegränsning
* Vinst och förlust "skärm"
